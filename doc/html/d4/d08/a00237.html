void Module_Decayer::Go ( int  argc,
char *  argv[] 
)

Top level function to call from main

A clock is started at the beginning of Go() to measure the total calculation time.

If too few arguments are given for the decayer, being "program module inputFile", then the printHelp() method is called and the program exits.

In a next step it'll checked if the filename points to an actual file. If yes, the calculation can start. If not, the program exits.

If the inputFile exists:

  • an SapphireInput object is created
  • the content of the inputFile is printed to std::cout via printInputFile()
  • the inputFile is read by ReadInputFile()
  • the parameters in the Sapphire Input Object are printed via printInputParameters()
  • the Run() method is called, passing the SapphireInput object by reference

At the end of the Go() method the clock is stopped and the total calculation time is printed.

void Module_Decayer::Run ( const SapphireInput &  input)

Declaration of the main function of the Decayer Module

At first the parameters needed to be initialized for the old Sapphire code are obtained from the SapphireInput object passed by reference to the Module_Decayer::Run() method.

Because of the "segmentation fault on more than 10 threads" bug, the system is asked via omp_get_max_threads() what the maximum numbers of threads are. If this value is larger than 10, then the maximum number of threads used for the calculation is fixed at 10 to prevent the "segmentation fault" bug. Once this issue is fixed, this can be removed.

In a next step, the input parameters for the Decayer are printed to std::ccout.

If the number of events is not a multiple of the chunkSize, then the number of remainder is calculated from the modulo. The number of maximum chunks is then derived from the ratio (events-remainder)/chunkSize.

For the Monte-Carlo decay, a unsigned int randomSeed[12] array is initialized.

Now a ProgressBar object is generated from Progressbar.h, initialized with numInChunk, which ist the number of decays in the current chunk.
