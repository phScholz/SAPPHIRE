void Module_CrossSection::Go ( int  argc,
char *  argv[] 
)

Top level function to call from main for the reaction Module.

Parameters
argcNumber of cmd line parameters of Sapphire.
argvThe array which contains the cmd line parameters of Sapphire.

At the beginning of the Go() method a clock is started for the measurement of the total calculation time.

It'll be checked, whether enough cmd line parameters are given, otherwise the printHelp() method is called.

If enough cmd line parameters are given, then it'll be checked if the inputFile given is an actual file.

If the inputFile can be found, a SapphireInput object is created, the inputFile will be printed out via SapphireInput::printInputFile() and the content will be read into the SapphireInput object via SapphireInput::ReadInputFile().

The next step is to test if a single reaction calculation, or a calculations for reactions given in a reactionFile need to be performed. If the reactionFile is a actual file, the calculation continues with Run(). If the reactionFile is not a regular file, then the calculations will be performed with RunSingleReaction(). This means, if both keywords are set, the reactionFile option will overwrite the reaction keyword.

If no regular inputFile is given in the first place, the code assumes that the cmd line parameter is a reactionString. In this case RunSingleReaction will be called by passing the cmd line parameter.

At the end of the Go() method, the clock is stopped and the total calculation time is printed to std::cout.
